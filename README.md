# XAI

### 5. XAI GlassBox (Explainability)
**Folder:** [`/xai-glassbox`](./xai-glassbox/)
* **Tech:** SHAP, Matplotlib, Streamlit
* **Description:** A "Plug-and-Play" dashboard for Model Interpretability.
* **The "Wow" Factor:** Provides both **Global** (Feature Importance) and **Local** (Waterfall plots) explanations. Includes a **What-If Simulator** that allows business users to adjust inputs (e.g., "Increase Income") and see the real-time impact on the risk score.
